# Qulix Systems (тестовое задание)

Приложение позволяет вводить, редактировать и удалять следующие данные: 

1.Работник
  - Идентификатор
  - Фамилия
  - Имя
  - Отчество
  - Дата приёма на работу
  - Должность
  - Компания

2.Компания
  - Идентификатор
  - Наименование
  - Организационно-правовая форма
  - Количество сотрудников *(заполняется автоматически)*
